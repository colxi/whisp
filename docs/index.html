WHISP
Secure modern Chat App with rich messages
LAUNCH
Whisp is an open source chat application that will work on your browser, desktop, and device indistinctly. It provides secure communications and a rich messaging experience (Text / Audio / Images) through a clean, modern, and intuitive user interface.

Unlike the modern messaging apps, Whisp does not run in the background keeping you online all the time, you will be online just meanwhile you keep the app open.

Whisp takes the essence of the most popular modern messaging apps, and transports it to the world of the traditional open & anonymous chats. Because of its decentralized condition, you are not bound to the Whisp official server, but to any available public server, managed by third parties. You can even, if you choose, create and set up your own private server.
Security
Whisp provides multiple layers of security. The first relies on the HTTPS protocol, which will encrypt by default all your sent messages and multimedia. It also provides the option to create secure Chat Rooms, where all the activity is encrypted with a private key token (using AES encryption).
Crossplatform
Because Whisp is powered by the Progressive Web Applications technology (PWA), it is capable of running outside the browser, in any device, as a standalone app. You can start the experience in your browser and eventually install it in your computer or device without needing to use an app store.
Open Source
The code is available without restrictions (MIT license) to everybody in Github. It's compact, heavily documented, and portable. You can download it and use it to create your own private server, or to modify the client to customize it to your personal needs.
